# Code Samples

This repository contains selected, self-written code samples
demonstrating architectural patterns, coding style and
problem-solving approaches.

## Context
The samples are inspired by real-world production systems.
Original projects were proprietary and cannot be shared.

## What this repository is NOT
- not a full application
- not runnable
- no business logic

## Topics covered
- Dependency Injection (Spring)
- Mapping & DTO separation
- Builder pattern
- Service layer design
- Frontend component & service structure
